## Using Ibis for transcoding SVGs used in org.pushingpixels projects

SVG content transcoded into Java2D-powered classes is used extensively across projects provided in the `org.pushingpixels` [namespace](https://github.com/kirill-grouchnikov). Here is a script you can use if you wish to transcode that SVG content locally on your machine:

```
IBIS_VERSION=...      // <-- put in the version of Ibis that you checked out, such as 1.0.0
FLAMINGO_VERSION=...  // <-- put in the version of Flamingo that you checked out, such as 5.3.00

java -cp ./flamingo/drop/$FLAMINGO_VERSION/flamingo-$FLAMINGO_VERSION.jar:./ibis/lib/*:./ibis/drop/$IBIS_VERSION/ibis-$IBIS_VERSION.jar org.pushingpixels.ibis.SvgBatchConverter ./flamingo/src/utest/svg/transcoded utest.svg.transcoded /org/pushingpixels/ibis/SvgTranscoderTemplateResizable.templ
java -cp ./flamingo/drop/$FLAMINGO_VERSION/flamingo-$FLAMINGO_VERSION.jar:./ibis/lib/*:./ibis/drop/$IBIS_VERSION/ibis-$IBIS_VERSION.jar org.pushingpixels.ibis.SvgBatchConverter ./flamingo-demo/src/org/pushingpixels/demo/flamingo/svg/tango/transcoded org.pushingpixels.demo.flamingo.svg.tango.transcoded /org/pushingpixels/ibis/SvgTranscoderTemplateResizable.templ
java -cp ./flamingo/drop/$FLAMINGO_VERSION/flamingo-$FLAMINGO_VERSION.jar:./ibis/lib/*:./ibis/drop/$IBIS_VERSION/ibis-$IBIS_VERSION.jar org.pushingpixels.ibis.SvgBatchConverter ./flamingo-demo/src/org/pushingpixels/demo/flamingo/svg/filetypes/transcoded org.pushingpixels.demo.flamingo.svg.filetypes.transcoded /org/pushingpixels/ibis/SvgTranscoderTemplateResizable.templ ext_

java -cp ./flamingo/drop/$FLAMINGO_VERSION/flamingo-$FLAMINGO_VERSION.jar:./ibis/lib/*:./ibis/drop/$IBIS_VERSION/ibis-$IBIS_VERSION.jar org.pushingpixels.ibis.SvgBatchConverter ./substance-demo/src/org/pushingpixels/demo/substance/main/check/svg/tango org.pushingpixels.demo.substance.main.check.svg.tango /org/pushingpixels/ibis/SvgTranscoderTemplateSubstance.templ
```
